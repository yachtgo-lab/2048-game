# 2048 Ranking PWA

구성:
- index.html
- manifest.json
- sw.js
- icon.svg
- icon-192.svg
- icon-512.svg

업데이트 방법:
1. GitHub 저장소에서 기존 파일을 이 파일들로 덮어쓰기
2. Commit changes 클릭
3. GitHub Pages 자동 재배포 대기
4. 휴대폰에서 새로고침 또는 앱 재실행

주의:
- 랭킹은 기기별 localStorage 저장입니다.
- 친구들과 공유되는 온라인 랭킹은 Firebase/Supabase 같은 DB가 필요합니다.
